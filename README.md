# RaTimer

Simple time tracking app demo for iOS8.

## Installation

Requires XCode 6.

Requires cocoapods, which can be installed with

sudo gem install cocoapods

Then, pods used by RaTimer can be installed with

pod install

## Usage

Simple iOS app for saving working hours on any number of projects. Displays total project hours for the current day, week, month, or year.
